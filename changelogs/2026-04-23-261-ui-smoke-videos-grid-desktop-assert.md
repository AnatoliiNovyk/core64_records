# 2026-04-23-261 UI Smoke Videos Grid Desktop Assert

## Як було
- Після переходу секції Video на 4 картки в ряд не було автоматичної UI-перевірки, що desktop-лейаут дійсно лишається 4-колонковим.

## Що зроблено
- У `scripts/ui-smoke.mjs` в `verifyPublicUi` додано desktop-контекст перевірки:
  - фіксований viewport `1920x1080`,
  - перевірка наявності `#videos-grid`,
  - перевірка класу `lg:grid-cols-4`,
  - перевірка фактичної кількості колонок через computed style (`gridTemplateColumns`) = 4.

## Що покращило/виправило/додало
- Вимога “4 відео в ряд на десктопі” закріплена автотестом і захищена від регресій.
- Будь-яка випадкова зміна класів/сітки секції Video тепер буде падати в UI smoke.
