# 2026-04-08-10 Admin Settings I18n Fix

## Як було

- У режимі Eng в секції Settings залишалися українські статичні підписи, заголовки та підказки.
- Частина текстів у полях була жорстко задана в `admin.html` без `data-i18n`.
- Атрибути `title` не перекладалися централізовано.

## Що зроблено

- У `admin.html` для блоку Settings додано `data-i18n`, `data-i18n-placeholder` і `data-i18n-title` для статичних заголовків, label-ів, кнопок, hint-текстів та option-ів.
- У `admin.js` розширено `applyAdminStaticTranslations()` підтримкою `data-i18n-title`.
- У словники `ADMIN_I18N.uk` і `ADMIN_I18N.en` додано нові ключі для всіх нових елементів локалізації секції Settings.
- Прогнано перевірки: diagnostics (`admin.html`, `admin.js`) та `npm run ui-smoke` (pass).

## Що покращило/виправило/додало

- EN-режим у Settings перестав змішувати мови для статичних UI-елементів.
- Уніфіковано локалізацію підказок (`title`) через той самий i18n-механізм.
- Зменшено ризик повторної появи hardcoded-рядків у Settings при подальших змінах.
