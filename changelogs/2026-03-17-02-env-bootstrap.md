# Changelog 2026-03-17 #02 - Backend env bootstrap

- У backend був лише `.env.example`.
- Реальні змінні середовища для міграцій/запуску API були відсутні.

## Що було зроблено
- Створено `backend/.env` на основі шаблону.
- Заповнено ключі: `PORT`, `NODE_ENV`, `DATABASE_URL`, `JWT_SECRET`, `CORS_ORIGIN`, `ADMIN_PASSWORD`.

## Що це покращило, виправило, додало
- Додало готове локальне оточення для запуску `migrate`, `seed` і `dev` серверу.
- Прибрало причину помилки "DATABASE_URL is empty".
- Підготувало проєкт до наступного кроку: перевірка доступності PostgreSQL і фактична міграція.
