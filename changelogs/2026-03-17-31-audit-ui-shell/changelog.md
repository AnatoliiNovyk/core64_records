# Changelog 2026-03-17 #31 - Підготовка UI журналу аудиту

## Як було
- Аудит уже зберігався в БД і читався API, але в адмінці не було окремої секції перегляду.

## Що було зроблено
- У `data-adapter.js` додано `getAuditLogs(limit)`.
- У `admin.html` додано:
  - пункт меню `Аудит`,
  - секцію `section-audit`,
  - контролі пошуку та ліміту,
  - контейнер `audit-list`.

## Що це покращило, виправило, додало
- Додало готову оболонку для візуального перегляду журналу дій.
- Зв’язало frontend з API журналу аудиту.
- Підготувало основу для оперативного моніторингу адмін-операцій.
