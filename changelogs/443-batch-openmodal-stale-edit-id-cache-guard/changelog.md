# 443 Batch - OpenModal stale edit id cache guard

як було:

- У `openModal` при edit-режимі, якщо `id` був заданий, але запис вже відсутній у `cache`, модалка відкривалась з порожнім об'єктом у гілці "Редагувати".

що зроблено:

- Додано перевірку `matchedItem` у `openModal`.
- Якщо запис за `editingId` не знайдено в `cache`, функція завершується до рендеру модалки.

що покращило/виправило/додало:

- Усунуто edge-case "phantom edit" при stale id.
- Підвищено передбачуваність edit-flow без впливу на штатні сценарії з валідним `id`.
