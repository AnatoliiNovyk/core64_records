# 419 Batch - File upload input tag guard

як було:

- У `handleFileUpload` була перевірка на існування/connected-стан аргументу, але не було явної валідації типу DOM-елемента.

що зроблено:

- Додано ранній guard: обробка продовжується лише якщо `input.tagName` існує і дорівнює `INPUT`.

що покращило/виправило/додало:

- Усунуто edge-case ризик неочікуваної обробки не-`INPUT` елемента при випадковому/некоректному виклику.
- Підвищено передбачуваність upload-флоу без зміни бізнес-логіки.
