# 492 Batch - Brand social icons inline SVG

## Як було

- У публічному фронтенді для SoundCloud/Instagram використовувались generic або fallback-іконки.
- Через обмеження бібліотеки іконок відображались «ліві» символи замість бренд-лого.

## Що зроблено

- Додано inline SVG бренд-іконки SoundCloud та Instagram у [app.js](app.js#L19) і використано їх у карточках артистів у [app.js](app.js#L118).
- У hero social-блоці сторінки [index.html](index.html#L221) замінено placeholder-іконки на inline SVG бренд-логотипи.
- Збережено існуючі класи/кольори hover для консистентного UI.

## Що покращило

- На фронтенді відображаються впізнавані соцлоготипи SoundCloud/Instagram без замінників.
- Візуальний стиль став узгодженим між карточками артистів і hero social-блоком.
- Прибрано залежність від резолву саме цих бренд-іконок через Lucide.
