# Changelog - 2026-03-17 - Audit latency indicator tooltip

Додано пояснювальний tooltip для індикатора latency у секції аудиту.
Tooltip тепер містить пороги (зелений/жовтий/червоний) та поточний статус.
Додано `aria-label` для кольорової точки індикатора, щоб стан був читабельним для assistive-технологій.

Файл: `admin.html`

- Для `#audit-avg-latency` додано базовий `title` з порогами:
- до 300ms — добре
- 301-800ms — помірно
- понад 800ms — повільно

Файл: `admin.js`

- Оновлено `updateAuditLatencyIndicator(avgMs)`:
- формує динамічний `title` з поточним станом і середнім значенням;
- встановлює `aria-label` для `#audit-latency-dot` відповідно до стану;
- підтримує fallback-стан `немає даних`.

Статичні перевірки:

- `admin.js`: без помилок
- `admin.html`: без помилок

Smoke API:

- `okPage=1; okTotal=5; okItems=1`

Кольоровий індикатор latency став самодокументованим у UI та більш доступним для користувачів із допоміжними технологіями.
