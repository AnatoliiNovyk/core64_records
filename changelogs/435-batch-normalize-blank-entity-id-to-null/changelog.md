# 435 Batch - Normalize blank entity id to null

як було:

- `normalizeEntityId` повертав `id.trim()` для рядків.
- Після переходу на nullish-логіку (`??`) порожній рядок міг залишатися як "defined" id, що потенційно переводило `openModal` у некоректний edit-режим.

що зроблено:

- Оновлено `normalizeEntityId`:
- не-рядкові значення повертаються без змін;
- рядок обрізається;
- порожній результат нормалізується в `null`.

що покращило/виправило/додало:

- Збережено коректну підтримку `id = 0`.
- Усунуто edge-case з порожнім строковим id у modal-flow.
- Підвищено консистентність нормалізації id без зміни штатної поведінки.
