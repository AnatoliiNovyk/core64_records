# 499 Batch - Hero social links open only in popup/new tab

## Як було

- Після доданого JS fallback один клік по Hero-соціконці міг відкривати ресурс і в поточному вікні, і в новій вкладці/вікні одночасно.

## Що зроблено

- Прибрано JS-обробник кліку з fallback-навігацією у [app.js](app.js).
- Для Hero-соцпосилань залишено тільки стандартну поведінку браузера через:
  - `target="_blank"`
  - `rel="noopener noreferrer"`

## Що це виправило

- Тепер клік по соціконках Hero відкриває ресурс тільки у новій вкладці/вікні.
- Поточна вкладка більше не повинна перенаправлятись одночасно.
