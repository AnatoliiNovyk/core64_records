як було:

що зроблено:

- Додано ранній guard: якщо `id` є рядком і після `trim()` порожній, функція завершується.

що покращило/виправило/додало:

- Усунуто edge-case виклики видалення з порожнім строковим id.
- Підвищено валідацію вхідних даних без змін штатної логіки для валідних id.
