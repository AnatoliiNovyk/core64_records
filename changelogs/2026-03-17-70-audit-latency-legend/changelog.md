# Changelog - 2026-03-17 - Audit latency visible legend

## Що зроблено
- Додано постійно видиму міні-легенду порогів latency у блоці статусу аудиту.
- Легенда показує три стани: `добре`, `помірно`, `повільно` з відповідними кольорами.
- Прибрано `aria-hidden` з точки індикатора, щоб `aria-label` зі станом затримки коректно озвучувався assistive-технологіями.

## Технічні зміни
- Файл: `admin.html`
  - У секції аудиту (`#section-audit`) після `#audit-avg-latency` додано inline-легенду:
    - зелений: `<= 300ms`
    - жовтий: `301-800ms`
    - червоний: `> 800ms`
  - Для `#audit-latency-dot` прибрано атрибут `aria-hidden="true"`.

## Валідація
- Статичні перевірки:
  - `admin.html`: без помилок
  - `admin.js`: без помилок
- Smoke API:
  - `okPage=1; okTotal=5; okItems=1`

## Результат
- Пороги latency тепер зрозумілі без hover/focus по tooltip.
- Індикатор стану став більш доступним для screen reader-користувачів.
