# Changelog 2026-03-17 #15 - TicketLink sync between admin and public

## Як було
- В адмін-формі подій не було окремого поля для посилання на квитки.
- Публічна сторінка орієнтувалась переважно на `ticketLink`, що могло бути крихким при різних форматах ключа.

## Що було зроблено
- У `admin.js` (форма події) додано поле `ticketLink` типу URL.
- У `app.js` додано уніфікацію читання лінка квитків через `event.ticketLink || event.ticket_link`.

## Що це покращило, виправило, додало
- Додало керування ticket URL з адмінки без ручних правок даних.
- Синхронізувало відображення кнопки "Квитки" на публічній сторінці з адмін-введенням.
- Підвищило сумісність даних між camelCase/snake_case полями.
