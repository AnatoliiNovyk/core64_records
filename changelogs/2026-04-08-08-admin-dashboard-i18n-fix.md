# 2026-04-08-08 Admin Dashboard I18n Fix

## Як було

- В англійському режимі адмінки Dashboard-блок лишався частково українським.
- Сайдбар перекладався на EN, але заголовок Dashboard, KPI-підписи, статус і блок "Остання активність" залишались у UK.
- Це створювало змішаний інтерфейс і порушувало консистентність локалізації.

## Що зроблено

- У [admin.html](admin.html) додано `data-i18n` для всіх статичних текстів Dashboard:
  - заголовок секції
  - KPI підписи
  - статус (label + value)
  - заголовок блоку активності
  - дефолтне welcome-повідомлення
- У [admin.js](admin.js) розширено словники `ADMIN_I18N` (UK/EN) новими ключами Dashboard.
- Прогнано перевірки:
  - diagnostics для `admin.html` і `admin.js`
  - `npm run ui-smoke` (passed)

## Що покращило/виправило/додало

- Англійський режим адмінки тепер повністю перекладає Dashboard-блок.
- Усунуто змішування мов у критичному стартовому екрані адмін-панелі.
- Підвищено цілісність UX локалізації між sidebar і main content.
