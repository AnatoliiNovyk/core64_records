# 428 Batch - Open modal id trim normalization

- як було:
  - `openModal` присвоював `editingId = id || null`, через що строковий id із пробілами міг пройти як truthy і відкрити модалку в edit-режимі.

- що зроблено:
  - Додано нормалізацію: для строкового `id` застосовується `trim()`.
  - `editingId` встановлюється з `normalizedId`, а порожнє значення переводиться в `null`.

- що покращило/виправило/додало:
  - Усунуто edge-case із пробільним id, який міг спричиняти неконсистентний edit-flow.
  - Підвищено надійність modal-state без зміни штатної логіки для валідних id.
