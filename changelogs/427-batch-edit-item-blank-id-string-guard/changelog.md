# 427 Batch - Edit item blank id string guard

як було:

- `editItem` відсікав `null/undefined` id, але порожній строковий id (`""` або пробіли) не блокувався.

що зроблено:

- Додано ранній guard: якщо `id` є рядком і `trim()` дає порожнє значення, функція завершується.

що покращило/виправило/додало:

- Усунуто edge-case виклики редагування з порожнім строковим id.
- Підсилено валідацію edit-flow без змін штатної поведінки для валідних id.
