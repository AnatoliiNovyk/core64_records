# Changelog 2026-03-17 #27 - Пошук та CSV-експорт звернень

У секції звернень не було текстового пошуку.
Не було можливості вивантажити відфільтровані звернення у CSV.

У `admin.html` додано поле пошуку `contacts-search`.
У `admin.html` додано кнопку `Експорт CSV`.
У `admin.js` додано:

- `getFilteredContacts()` (статус + дата + пошук),
- `exportContactsCsv()` з експортом відфільтрованого набору,
- CSV escaping для безпечного формування файлу.

Додало швидкий оперативний пошук звернень.
Додало експорт для передачі/звітності поза адмінкою.
Підвищило зручність роботи з великим потоком заявок.
